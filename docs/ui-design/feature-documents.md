## 1. 设计原则

### 1.1 设计语言

**设计理念**
- **简洁优先**：去除冗余元素，聚焦内容创作
- **沉浸式编辑**：最大化编辑区域，减少干扰
- **即时反馈**：操作结果快速可见，状态清晰透明
- **渐进增强**：核心功能优先，高级功能按需展示

**视觉风格**
- **现代简约**：扁平化设计，清晰的信息层级
- **专业工具**：类似 Notion/Obsidian 的专业编辑体验
- **舒适阅读**：优秀的排版和间距，减少视觉疲劳
- **品牌一致**：遵循 local-rag 整体设计规范

### 1.2 设计系统

**组件库**
- 基于 **Naive UI** 组件库
- 自定义主题配置（深色/浅色模式）
- 统一的间距、圆角、阴影规范

**色彩规范**
- **主色调**：科技蓝（#2080f0）- 主要操作、按钮
- **辅助色**：成功绿（#18a058）、警告橙（#f0a020）、错误红（#d03050）
- **中性色**：灰度色阶（#ffffff → #1a1a1a）
- **深色模式**：自动切换深色主题

**字体规范**
- **字体族**：-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial
- **字号层级**：12px / 14px / 16px / 18px / 20px / 24px
- **行高**：1.5 - 1.8（编辑器），1.2 - 1.4（界面）
- **字重**：Regular (400) / Medium (500) / Semibold (600)

**间距规范**
- **基础单位**：4px
- **组件间距**：8px / 16px / 24px / 32px
- **页面边距**：24px（桌面端），16px（移动端）
- **内容间距**：段落 16px，章节 24px

## 3. 核心页面设计

### 3.1 笔记列表页

#### 布局结构

```
┌──────────────────────────────────────────────────────────────────────┐
│  顶部操作栏                                                          │
│  ┌─────────────────────┐  ┌──────────────┐  ┌──────────────────┐   │
│  │ 搜索框 [🔍]         │  │ 标签筛选 ▼   │  │ [+ 新建笔记]     │   │
│  └─────────────────────┘  └──────────────┘  └──────────────────┘   │
├──────────────────────────────────────────────────────────────────────┤
│  标签栏（横向滚动）                                                  │
│  [全部] [javascript (12)] [python (8)] [tutorial (15)] [...]        │
├──────────────────────────────────────────────────────────────────────┤
│  笔记列表（卡片视图 / 列表视图切换）                                 │
│  ┌────────────────────────────────────────────────────────────┐     │
│  │  笔记标题                         [标签1] [标签2]   ⋮  更多  │     │
│  │  笔记内容预览（前 100 字）                                   │     │
│  │  📅 2026-03-28 14:30  ✏️ 2026-03-28 15:20                    │     │
│  └────────────────────────────────────────────────────────────┘     │
│  ┌────────────────────────────────────────────────────────────┐     │
│  │  另一篇笔记标题...                                           │     │
│  └────────────────────────────────────────────────────────────┘     │
│                                                                      │
│  < 上一页  1 2 3 ... 10  下一页 >                                    │
└──────────────────────────────────────────────────────────────────────┘
```

#### 组件清单

1. **顶部操作栏**
   - 搜索框：支持实时搜索，输入时过滤笔记
   - 标签筛选下拉框：多选标签（AND 逻辑）
   - 新建笔记按钮：主操作按钮，固定位置

2. **标签栏**
   - 横向滚动标签列表
   - 显示标签名称和笔记数量
   - 点击标签快速筛选
   - 支持 "全部" 选项

3. **笔记卡片**
   - 标题：加粗，可点击进入编辑器
   - 预览：前 100 字内容预览
   - 标签：显示关联的标签
   - 时间戳：创建时间和更新时间
   - 操作按钮：编辑、删除、更多（...）

4. **分页组件**
   - 标准分页控件
   - 显示总数和当前页

#### 交互说明

**搜索交互**
- 输入关键字实时过滤笔记
- 搜索结果高亮关键字
- 支持搜索标题和内容

**标签筛选交互**
- 点击标签栏的标签快速筛选
- 使用标签筛选下拉框选择多个标签
- 标签组合使用 AND 逻辑（同时包含所有选中标签）

**排序交互**
- 默认按更新时间降序
- 支持切换排序方式：创建时间、更新时间、标题
- 支持升序/降序切换

**视图切换**
- 卡片视图：预览内容，适合浏览
- 列表视图：紧凑显示，适合大量笔记

**操作交互**
- 点击卡片标题/预览区域 → 进入编辑器
- 点击编辑按钮 → 进入编辑器
- 点击删除按钮 → 弹出确认对话框
- 点击更多按钮 → 下拉菜单（重命名、复制、导出等未来功能）

#### 空状态

```
┌────────────────────────────────────────────────────────────┐
│                                                            │
│                      📝                                    │
│                                                            │
│                  还没有笔记                                │
│                                                            │
│           [+ 新建笔记] 开始创建第一条笔记                   │
│                                                            │
└────────────────────────────────────────────────────────────┘
```

#### 响应式设计

**桌面端**（≥ 768px）
- 两列布局：左侧标签栏（固定），右侧笔记列表
- 卡片视图每行 2-3 个卡片

**平板端**（768px - 1024px）
- 单列布局
- 卡片视图每行 1-2 个卡片

**移动端**（< 768px）
- 单列布局
- 卡片视图每行 1 个卡片
- 标签栏横向滚动
- 隐藏次要操作按钮

### 3.2 笔记编辑器页

#### 布局结构

```
┌──────────────────────────────────────────────────────────────────────┐
│  编辑器工具栏                                                        │
│  [← 返回]  标题输入框                         💾 已保存  ⋮ 更多     │
├──────────────────────────────────────────────────────────────────────┤
│  元数据区域（可折叠）                                                │
│  标签: [标签1] [标签2] [+ 添加标签]                                 │
├──────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  Markdown 编辑器 (md-editor-v3)                                      │
│                                                                      │
│  ┌────────────────────┬─────────────────────────────────────────┐  │
│  │  编辑模式          │  预览模式                               │  │
│  │                    │                                         │  │
│  │  # 标题            │  标题                                   │  │
│  │                    │                                         │  │
│  │  正文内容...       │  正文内容...                            │  │
│  │                    │                                         │  │
│  │  - 列表项          │  • 列表项                               │  │
│  │                    │                                         │  │
│  └────────────────────┴─────────────────────────────────────────┘  │
│                                                                      │
│  编辑器工具栏（md-editor-v3 内置）                                   │
│  [B] [I] [H1] [H2] [链接] [图片] [代码] [列表] [引用] [...]        │
│                                                                      │
└──────────────────────────────────────────────────────────────────────┘
```

#### 组件清单

1. **顶部工具栏**
   - 返回按钮：返回笔记列表
   - 标题输入框：大字号，内联编辑
   - 保存状态：显示 "已保存" / "保存中..." / "未保存"
   - 更多菜单：删除、复制、导出等

2. **元数据区域**
   - 标签编辑：标签列表 + 添加标签输入框
   - 可折叠：点击标题折叠/展开
   - 创建/更新时间：显示只读信息

3. **Markdown 编辑器**
   - 分屏模式：左侧编辑，右侧预览
   - 全屏模式：隐藏其他元素，专注编辑
   - 工具栏：常用格式按钮
   - 状态栏：字数统计、光标位置

4. **底部操作栏**（可选）
   - 保存按钮：手动保存（Ctrl+S）
   - 向量化状态：显示向量化进度
   - 帮助按钮：快捷键提示

#### 交互说明

**编辑交互**
- 实时预览：输入时同步渲染预览
- 快捷键：支持 Markdown 常用快捷键
- 自动保存：停止输入 30 秒后自动保存

**保存状态**
- **已保存**：绿色提示，内容已保存
- **保存中...**：加载动画，正在保存
- **未保存**：黄色提示，有未保存的更改

**标签编辑**
- 点击标签移除：从笔记中删除标签
- 输入标签名：按回车或逗号添加标签
- 自动补全：显示已有标签列表

**编辑器模式**
- **分屏模式**：左右分栏，编辑和预览并排
- **编辑模式**：只显示编辑器，隐藏预览
- **预览模式**：只显示预览，隐藏编辑器
- **全屏模式**：隐藏所有 UI 元素，只保留编辑器

**向量化提示**
```
┌────────────────────────────────────────────────────────────┐
│  ⏳ 向量化中... (2/3 chunks 已完成)                         │
└────────────────────────────────────────────────────────────┘
```

#### 响应式设计

**桌面端**（≥ 1024px）
- 分屏模式：左右各 50%
- 固定工具栏

**平板端**（768px - 1024px）
- 分屏模式：上下布局
- 可折叠工具栏

**移动端**（< 768px）
- 单屏模式：编辑/预览切换
- 底部工具栏：固定在底部
- 标签编辑：弹窗形式

## 5. 响应式设计

### 5.1 断点设置

| 断点名称 | 屏幕宽度 | 目标设备 | 布局调整 |
|---------|---------|---------|---------|
| 移动端 | < 768px | 手机 | 单列布局 |
| 平板端 | 768px - 1024px | 平板、小平板 | 双列布局 |
| 桌面端 | ≥ 1024px | 桌面、大平板 | 多列布局 |

### 5.2 布局适配

#### 笔记列表页

**移动端**（< 768px）
- 单列布局
- 隐藏次要操作
- 标签栏横向滚动
- 卡片最小高度 120px

## 8. 性能优化

### 8.1 图片优化

#### 策略
- 使用 WebP 格式（降级 JPEG）
- 响应式图片（srcset）
- 懒加载（Intersection Observer）
- 缩略图和原图分离

#### 实现
```html
<img
  src="thumbnail.webp"
  srcset="
    thumbnail-320w.webp 320w,
    thumbnail-640w.webp 640w,
    thumbnail-1280w.webp 1280w
  "
  sizes="(max-width: 768px) 100vw, 50vw"
  loading="lazy"
  alt="笔记预览图"
/>
```

#### 列表懒加载
- 分页加载（每页 20 条）

- 使用 Intersection Observer

## 9. 设计规范总结

### 9.1 核心设计原则

1. **简洁优先**：去除冗余，聚焦内容
2. **即时反馈**：操作结果快速可见
3. **渐进增强**：核心功能优先
4. **响应式设计**：适配所有设备
5. **无障碍友好**：支持键盘导航

### 9.2 关键设计决策

1. **为什么选择 Naive UI？**
   - Vue 3 原生支持
   - 组件完善，文档详细
   - 支持深色模式
   - 性能优秀

2. **为什么使用卡片视图？**
   - 信息层级清晰
   - 适合快速浏览
   - 移动端友好
   - 视觉吸引力强

3. **为什么采用分屏编辑器？**
   - 实时预览，所见即所得
   - 符合 Markdown 编辑习惯
   - 提升编辑效率

4. **为什么自动保存而不是手动保存？**
   - 防止数据丢失
   - 降低操作负担
   - 符合现代工具习惯

- 线框图（Figma/Sketch）

- 设计稿（标注、切图）
- 图标资源（SVG/PNG）
- 颜色和字体规范

**验收标准**
- 视觉还原度 ≥ 95%
- 交互流程完整
- 响应式适配
- 无障碍合规
