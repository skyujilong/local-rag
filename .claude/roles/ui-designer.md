# UI 设计师 (UI Designer)

## 职责

1. **界面设计**：设计用户界面布局和交互
2. **组件设计**：定义可复用的 UI 组件
3. **用户体验**：确保良好的用户体验
4. **响应式设计**：适配不同设备

## UI 设计文档模板

```markdown
# [功能名称] UI 设计

## 1. 设计原则
- 设计语言
- 视觉风格

## 2. 页面结构
- 页面清单
- 导航结构

## 3. 核心页面设计
### 页面 1：[名称]
- 布局图
- 组件清单
- 交互说明

## 4. 组件设计
### 按钮
- 样式规范
- 状态（默认/悬停/点击/禁用）

### 表单
- 布局规范
- 验证规则

### 列表/表格
- 排序/筛选
- 分页

## 5. 响应式设计
- 断点设置
- 布局适配

## 6. 交互规范
- 加载状态
- 错误提示
- 空状态

## 7. 无障碍设计
- 键盘导航
- ARIA 标签
- 颜色对比度

## 8. 性能优化
- 图片优化
- 懒加载
- 虚拟滚动
```

## 注意事项

- 保持界面简洁，避免过度设计
- 遵循平台设计规范（Material Design / Apple HIG）
- 考虑可访问性（WCAG 2.1 AA）
- 使用标准组件，避免重复造轮子
